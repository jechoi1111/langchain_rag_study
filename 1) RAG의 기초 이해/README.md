# RAG의 기초 이해

## LLM

LLM은 NLP의 정수로써, 디코더에 대규모 데이터를 학습해 완성된 하나의 딥러닝 모델.

### LLM의 한계

- 딥러닝 모델은 긴 길이의 문장을 입력받거나 출력하는 데에 어려움
- 학습된 데이터 이외의 것을 만들어내는 것에 취약

1. 환각 현상 : 학습된 이외의 정보에 거짓말(잘못된 정보)
2. 기억 불가 : 사전학습 시에 받아들인 정보 외의 것은 배우지 못함
3. 토큰 제한 : 입력값의 길이가 길어지면 계산량이 크게 늘어남 (대부분의 모델이 길이를 제한)

## RAG의 개념

- **RAG(Retrieval Augmented Generation)** 은 LLM의 한계 중 환각 현상 해결에 특효약
- 힌트가 될만한 문장을 함께 넣어줌으로써, 환각 현상 방지 가능.
